# Javascript-Quiz2
A fun JavaScript Quiz 

This is a Quiz about Javascript and made with Javascript.  
The Quiz begins with a prompt that prompts the user to select an answer to a question from an array of possible answers.
Once an answer is selected, a prompt lets the user know whether the answer was correct or inccorrect.
The Quiz is timed with a 10 second timer.
If an answer is not chosen within the alotted time, the next question will automatically prompt.
A score is also generated with the tally increasing with each correct answer within the allowed time.
The score is saved at the end of the quiz.
